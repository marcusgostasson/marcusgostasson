Hi, I’m Marcus Göstasson

I’m a final-year Software Development student at Kristianstad University, graduating in June 2026. My studies have given me a broad foundation in computer science, including object-oriented programming, databases, algorithms and data structures, computer security, operating systems, data communication, full stack development, machine learning, big data analytics, and software engineering.
